# Windows システムコマンド

## 基本コマンド
- `dir` - ファイル一覧表示（`ls` の代わりに）
- `cd` - ディレクトリ変更
- `mkdir` - ディレクトリ作成
- `copy` - ファイルコピー
- `move` - ファイル移動
- `del` - ファイル削除
- `rmdir` - ディレクトリ削除

## 検索・フィルタリング
- `findstr` - テキスト検索（`grep` の代わりに）
- `where` - ファイル検索（`which` の代わりに）

## Git操作（共通）
- `git status` - 作業ツリー状況確認
- `git add` - ステージング
- `git commit` - コミット
- `git diff` - 差分表示

## プロジェクト固有
- 開発サーバー起動時は `chcp 65001` でUTF-8エンコーディング設定済み
- MSYS_NT-10.0環境での実行
